
## How to Run on Local
* npm install
* npm run dev

## Details about features
* When no filter or search condition is given, app will render all the products
* search and filter works independently (only those items will be shown which satisfy both filter and search condition ), (search matches with products which starts with value in the input field)
* search is not case-sensitive
* Images are set to load lazily
* Item description opens in a different route using react-router-dom
